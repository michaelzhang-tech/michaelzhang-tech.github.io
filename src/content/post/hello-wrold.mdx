## 为什么选择 Astro？

Astro 是一个现代化的静态站点生成器，具有以下优势：

- **零 JavaScript 默认**：页面加载极快
- **内容优先**：完美支持 Markdown 和 MDX
- **群岛架构**：只在需要的地方添加交互性
- **优秀的 SEO**：预渲染的 HTML 对搜索引擎友好
